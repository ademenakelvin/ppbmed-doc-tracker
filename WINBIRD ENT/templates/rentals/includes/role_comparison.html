Role Comparison

Staff vs Admin access

Admin {% if request.user.role == 'ADMIN' %} Current role {% endif %}

Admins oversee approvals, stock control, reports, and broader system management.

  • Can confirm and cancel bookings
  • Can add and edit rental items, prices, and inventory
  • Can create staff accounts and manage reports
  • Can monitor staff activity and payment follow-up
Staff {% if request.user.role == 'STAFF' %} Current role {% endif %}

Staff focus on booking entry, customer handling, payments, dispatch, and returns.

  • Can create bookings and record payments
  • Can view inventory, bookings, and alerts
  • Can mark items out and returned during fulfilment
  • Cannot approve bookings or manage catalog setup
